# AOL Software Engineering
AI chatbot
Web Application
for food based recommendation, in search of diet, mood and price.

# Requirements
Install Vite and Node.jsAdd commentMore actions
Node.js v18+
npm 

# How to run
```Bash
cd superapp-food-delivery
npm install vite

cd backend
npm install
npx nodemon index.js

cd vite-project
npm run dev
```
